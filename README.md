Online Store 

Developed a dynamic Online Store Web Application using Java (Servlet & JSP), HTML, CSS, and JavaScript. 
The project simulates a real-world e-commerce system with user interaction and backend processing.

Key features include:
User-friendly interface for browsing and purchasing products
Dynamic data handling using Servlets and JSP
Form handling, validation, and session management
Interactive frontend with JavaScript
Clean and responsive UI using HTML & CSS

This project demonstrates strong understanding of full-stack web development, including backend logic, request-response handling, and frontend integration.
